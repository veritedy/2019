CSS

CSS(Cascading Style Sheets)는 HTML을 꾸며주는 디자인 언어 스타일시트입니다.


CSS 선택자

    • 하나의 태그에 하나만 사용 할 수 있습니다.
    • "#"으로 표시합니다.
  • class 선택자
    • 여러개의 태그를 식별하기 위한 선택자입니다.

    • "."으로 표시합니다.

float으로 인한 영역 깨짐 방지법

float을 사용하면 height 값을 0으로 인식합니다.

  1. 깨지는 영역에 똑같이 float을 사용합니다. (X) 모든 영역에 float을 사용하게 되고 정확히 깨지는 영역을 알 수가 없습니다.
  2. float의 성질을 차단하는 clear;both를 사용합니다. (X) 깨지는 영역을 정확히 확인 할 수 없습니다.
  3. float을 사용한 부모 박스한테 overflow:hidden을 사용합니다. (O)
  4. claerfix 속성을 만들어서 사용합니다. (O)

컬러를 표현하는 방법

HTML에서 색을 표현하는 방법에는 색이름으로 표현하는방법, HEX값으로 표기하는방법, RGB값으로 표기하는 방법이 있습니다.

  • 색상명 : color-name;
  • HEX값 : #000000; - #RRGGBB형식의 16진수 색상값
  • RGB값 : rgb(0,0,0); - 각 변수에 0~255 사이의 정수 또는 백분율값
  • RGBA값 : rgba(0,0,0,0); - RGB값에 alpha채널(투명도)이 더해진 값
  • HSL값 : hsl(360%,100%,100%); - 색상(0~360%), 채도(0~100%), 명도(0~100%)
  • HSLA값 : hsla(0%,0%,0%,0); - HSL값에 alpha채널(투명도)이 더해진 값
  • Opacity : opacith: 0; - 색상에 대한 투명도로 0.0~1.0사이 값

단위를 표현하는 방법

-절대단위 (in, cm, mm, pt, pc) : 정확한 기준에 의해 고정된 단위, 출력 매체에 쓰기 적절합니다.

-상대단위 (px, em, ex, %, rem, vw, vh, vmin, vmax) : 어떤 기준을 통해 상대적으로 변할 수 있는 단위로, 다양한 매체에 쓰기 적절합니다.

  • px : 모니터의 1dot이 1pixel을 의미하며, pixel은 해상도에 따라서 상대적인 크기를 갖습니다.
  • em : 부모 요소의 대문자 너비를 기준으로 상대적인 값을 갖습니다.
  • ex : 부모 요소의 소문자 너비를 기준으로 상대적인 값을 갖습니다.
  • % : 부모 요소의 길이를 기준으로 상대적인 값(%)을 갖습니다.
  • rem : 루트(root em) 요소의 크기를 기준으로 상대적인 값을 갖습니다.
  • vw : 뷰포트(viewport) 브라우저 가로 크기를 기준으로 상대적인 값을 갖습니다.
  • vh : 뷰포트(viewport) 브라우저 세로 크기를 기준으로 상대적인 값을 갖습니다.
  • vmin : 뷰포트(viewport) 브라우저 가로 크기 또는 세로 크기 중 작은 값을 기준으로 상대적인 값을 갖습니다.
  • vmax : 뷰포트(viewport) 브라우저 가로 크기 또는 세로 크기 중 큰 값을 기준으로 상대적인 값을 갖습니다.
  • in : in은 인치를 의미합니다.
  • cm : cm은 센치미터를 의미합니다.
  • mm : mm는 밀리미터를 의미합니다.
  • pt : pt는 포인트를 의미합니다.
  • pc : .pc는 파이카를 의미합니다.

CSS를 사용하는 방법

  • 외부 선언 (external css) - 외부 파일로 연결하는 방법
  • 내부 선언 (internal css) - 내부 파일에서 사용하는 방법
  • 요소 선언 (inline css) - 요소에 직접 사용하는 방법

절대경로 상대경로

박스 요소의 바깥쪽 여백을 설정합니다.

  • 절대경로 : 모든 디렉토리가 포함된 모든 경로의 주소
  • 상대경로 : 이미지를 삽입할 HTML 문서를 기준으로 경로를 표현하는 주소
  • / : 루트 - 가장 최상의 디렉토리로 이동합니다.(Web root)
  • ./ : 현재 위치 - 파일이 현재 디렉토리를 의미합니다.
  • ../ : 현재 위치의 상단 폴더 - 상위 디렉토리로 이동합니다.
  • '../../' : 현재 위치의 상단 상단 폴더 - 두단계 상위 디렉토리로 이동합니다.

컨텐츠 요소를 안보이게 하는 방법

  • display : none; --display: block; (영역X) (애니X)
  • opacity : 0 -- opacity: 1; (영역O) (애니O)
  • visibility : hidden -- visibility : visible; (영역O) (애니X)
  • IR 기법 : width:0, height:0;

미디어쿼리(Mediaquery)

미디어 쿼리는 화면 크기에 따른 각각의 속성 값을 지정하여 여러가지 화면을 구성하는 기술입니다.

  • @media [only/not] [media type] [and/,] (조건문){실행문}
  • @media : 미디어쿼리 시작을 의미합니다.
  • [only/not] : only는 미디어쿼리를 지원하는 브라우저에서만 해석하고 not은 반대를 의미합니다. 미디어 쿼리 구문을 해석하라는 명령어입니다. (생략가능)
  • [media type] : 미디어쿼리를 해석해야 할 대상을 나타냅니다. (생략가능)
    • all : 모든 미디어 유형
    • screen : 컴퓨터 화면
    • print : 인쇄 장치
    • handheld : 휴대용 장비
    • tv : 텔레비전
    • aural : 음성출력기
    • beaille : 점자 출력기
    • projection : 프로젝트
    • tty : 고정폭을 가지는 텔레타이프
  • [and/,] : and는 둘다 조건에 맞아야 하고, ','는 하나만 맞아도 실행합니다. 앞과 뒤에 조건을 나타냅니다.
  • 조건문 : 해당 조건을 설정합니다.
  • 실행문 : 조건에 따른 실행을 설정합니다.


CSS Reference

background

백그라운드 배경 속성 값을 설정합니다.

background-images

백그라운드 이미지를 설정합니다.

  • background-images : none; -백그라운드 이미지 속성을 설정하지 않습니다.
  • background-images : url(이미지 경로) : 백그라운드 이미지 설정합니다.

    • background-attachment

    백그라운드 이미지 고정여부 설정합니다.

    • background-attachment: scroll(기본값)
    • background-attachment: fixed; - 배경이미지를 고정하고 컨텐츠만 움직입니다.
    • background-attachment: local; - 배경이미지와 콘텐츠가 같이 움직입니다.

    background-position

    백그라운드 위치 설정합니다.

    • background-position: left top;
    • background-position: left center;
    • background-position: left bottom;
    • background-position: right top;
    • background-position: right center;
    • background-position: right bottom;
    • background-position: center top;
    • background-position: center center;
    • background-position: center bottom;
    • background-position: 10px 20px;
    • background-position: 10% 20%;
    • background-position: 10% 20%, 10px 20px, center top;

    background-repeat

    백그라운드 이미지 반복 여부를 설정합니다.

    • background-repeat: repeat(기본) - 이미지를 x축 y축으로 반복 설정합니다.
    • background-repeat: repeat-x - 이미지를 x축으로 반복 설정합니다.
    • background-repeat: repeat-y - 이미지를 y축으로 반복 설정합니다.
    • background-repeat: no-repeat - 이미지를 반복하지 않고 하나만 나오게 합니다.

      • background-size

      백그라운드 이미지 사이즈를 설정합니다.

      • background-size: length(단위)
      • background-size: auto(기본값)
      • background-size: contain - 이미지를 요소의 크기에 맞게 설정합니다. 가로와 세로 비율을 유지하기 때문에 공백이 생길 수 있습니다.
      • background-size: cover - 이미지를 요소의 크기에 맞게 설정합니다. 공백 없이 화면 크기에 맞추기 때문에 이미지가 왜곡될 수 있습니다.

      background-blend-mode

      백그라운드 이미지의 블랜드 모드를 설정합니다.


      background-color

      백그라운드 색을 설정합니다.

      • background-color: transparent(기본) - 투명하게 설정
      • background-color: color; - 백그라운드 컬러값을 설정

      border

      테두리를 만드는 속성입니다. 위, 오른쪽, 아래, 왼쪽 선을 한 번에 정합니다.


      border-color

      선의 색을 정합니다. 기본값은 꾸미려는 요소의 색(color)입니다.

      • color : 색상 키워드, 16진수 값, rgb 값등의 css 컬러 속성 값을 지정한다.
      • transparent : 투명을 지정한다.
      • inherit : 부모 요소로부터 값을 상속 받는다.

      border-style

      선의 모양을 정합니다. 기본값은 none입니다.

      • hidden : none 속성값과 같다.
      • dotted, dashed, solid, double, groove, ridge, inset, outset
      • inherit : 부모 요소로부터 값을 상속 받는다.

      border-width

      선의 두께를 정합니다. 기본값은 medium입니다.

      • length : 부동 소수점 숫자 뒤에 절대 단위 지정자( cm , mm , in , pt , pc ) 또는 상대 단위 지정자( em , ex , px )가 오는 값을 지정한다.
      • initial; - 기본값으로 설정합니다.
      • inherit : 부모 요소로부터 값을 상속 받는다.

      border-top

      상단 테두리 색상, 너비, 스타일을 설정합니다.


      border-top-color

      상단 테두리 색상을 설정합니다.


      border-top-width

      상단 테두리 너비를 설정합니다.


      border-top-style

      상단 테두리 영역의 스타일을 설정합니다.


      border-right

      오른쪽 테두리의 속성값을 설정합니다.


      border-right-color

      오른쪽 테두리의 색상을 설정합니다.


      border-right-width

      오른쪽 테두리의 너비를 설정합니다.


      border-right-style

      오른쪽 테두리의 스타일을 설정합니다.


      border-bottom

      하단 테두리 색상, 너비, 스타일을 설정합니다.


      border-bottom-color

      하단 테두리의 색상을 설정합니다.


      border-bottom-width

      하단 테두리 너비를 설정합니다.


      border-bottom-style

      하단 테두리 영역의 스타일을 설정합니다.


      border-left

      왼쪽 테두리의 속성값을 설정합니다.


      border-left-color

      왼쪽 테두리의 색상을 설정합니다.


      border-left-width

      왼쪽 테두리의 너비를 설정합니다.


      border-left-style

      왼쪽 테두리의 스타일을 설정합니다.


      border-image

      요소 주위의 테두리로 사용할 이미지를 설정할 수 있습니다.


      border-image-outset

      테두리 이미지 영역이 테두리 상자를 넘어서 확장되는 양을 설정합니다.


      border-image-repeat

      테두리 이미지의 스케일을 조정하는 방법을 지정합니다.


      border-image-slice

      테두리 이미지를 분할하는 방법을 설정합니다.


      border-image-source

      테두리로 사용할 이미지의 경로를 설정합니다.


      border-image-width

      테두리로 사용될 이미지의 너비를 지정합니다.


      border-radius

      벤더 프리픽스(Vender Prefix)

      html3/css3으로 넘어가는 과정에서 몇몇의 css3 기능들이 웹 표준이 아니여서 다른 브라우저에서 제대로 표현되지 않을 때 사용하는 방법이다.
      Vendor는 "공급업체"이고, Perfix는 "접두사"이다. 즉, 벤더 프리픽스(Vender Prefix)는 '공급업체 접두사'라고 정의내릴 수 있다.
      브라우저 접두어를 앞쪽에 쓰고, 마지막에 표준 속성을 쓴다.

      • webkit - 크롬/사파리/안드로이드/iOs
      • moz - 파이어폭스/모질라
      • o - 오페라
      • ms - 인터넷 익스플로러

      해상도에 따른 이미지 처리하는 방법

      <img>태그를 이용하는 방법 : srcset과 sizes 속성을 추가하여 반응형 이미지 처리
      - srcset : srcset을 지원하는 브라우저는 이 값을 읽어 현재의 화면 상태에 맞는 적절한 이미지를 불러옵니다.
      - sizes: 미디어쿼리의 중단점별로 필요한 이미지의 정보를 추가로 제공함으로 브라우저로 하여금 현재의 상태에 맞는 이미지를 불러옵니다. 

      <img src="assets/img/blog_img01_1x.jpg"
srcset="assets/img/blog_img01_1x.jpg 1x,
assets/img/blog_img01_2x.jpg 2x,
assets/img/blog_img01_3x.jpg 3x,
assets/img/blog_img01_4x.jpg 4x" alt="블로그 이미지">

      background를 이용하는 방법 : css에 미디어쿼리를 추가하여 해상도에 따라 각 해상도에 맞는 사이즈의 이미지를 보여주도록 설정합니다.

      .img {background: url(img_1x.jpg);}

      3D transform 속성

      CSS3에서는 transform 속성을 사용하여 HTML 요소의 모양, 크기 위치 등을 입체적으로 변형시킬 수 있다.

      • transform - 요소에 2D 또는 3D 변형(transform)을 적용한다.
      • transform-origin - 요소에 변형(transform)을 적용하는 변환 중심을 설정한다.
      • transform-style - 요소에 변형을 적용할 때 그 변환이 자식(child) 요소들에게도 적용될지 안 될지를 설정한다.
      • perspective - 3D 요소에 원근감을 표현할 때 사용할 픽셀 수를 설정한다.
      • perspective-origin - 3D 요소에 원근감을 표현할 때 사용할 기준 축을 설정한다.
      • backface-visibility - 요소의 앞면만을 표현하고, 뒷면을 표현할지 안 할지를 설정한다.

      3D transform 메소드

      3D 변형(transform)을 위해 제공되는 메소드(method)는 다음과 같다.

      • matrix3d(n×16) - 4x4 행렬을 이용한 16개의 매개변수로 모든 3D 변형 메소드를 한 번에 설정한다.
      • rotate3d(x,y,z,angle) - 해당 요소를 주어진 각도만큼 x축, y축과 z축을 기준으로 회전시킨다.
      • rotateX(angle) - 해당 요소를 주어진 각도만큼 x축을 기준으로 회전시킨다.
      • rotateY(angle) - 해당 요소를 주어진 각도만큼 y축을 기준으로 회전시킨다.
      • rotateZ(angle) - 해당 요소를 주어진 각도만큼 z축을 기준으로 회전시킨다.
      • translate3d(x,y,z) - 현재의 위치에서 해당 요소를 주어진 x축, y축과 z축의 거리만큼 이동시킨다.
      • translateX(x) - 현재의 위치에서 해당 요소를 주어진 x축의 거리만큼 이동시킨다.
      • translateY(y) - 현재의 위치에서 해당 요소를 주어진 y축의 거리만큼 이동시킨다.
      • translateZ(z) - 현재의 위치에서 해당 요소를 주어진 z축의 거리만큼 이동시킨다.
      • scale3d(x,y,z) - 해당 요소의 크기를 주어진 배율만큼 x축, y축과 z축 방향으로 늘리거나 줄인다.
      • scaleX(x) - 해당 요소의 x축 크기를 주어진 배율만큼 늘리거나 줄인다.
      • scaleY(y) - 해당 요소의 y축 크기를 주어진 배율만큼 늘리거나 줄인다.
      • scaleZ(z) - 해당 요소의 z축 크기를 주어진 배율만큼 늘리거나 줄인다.
      • perspective(n) - 3D 요소에 원근감을 표현할 때 사용할 픽셀 수를 설정한다.
